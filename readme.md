## Setup
### Prerequisites

1. Install [Node.js v8 or greater][node]

1. Clone this repository:

        git clone https://github.com/mickey-ichi/sample-server.git

1. Change directory in to the project:

        cd sample-server

[node]: https://nodejs.org/

## How to run
1. Install dependencies

        npm install

1. Start project

        npm start
